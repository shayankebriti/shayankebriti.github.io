FractMorph

Deformable registration Transformer Reproducible code

Multi-domain transformer framework for deformable medical image registration.
Preprint available; journal revision ongoing (Medical Image Analysis, IF 11.8).

Highlights

  • Problem: robust alignment across modalities under anatomical variability and frequency-domain mismatch.
  • Contribution: multi-domain modelling with spatial + fractional Fourier representations; end-to-end training and evaluation pipeline with ablations.
  • Evidence: competitive benchmark results; open-source implementation enabling reproduction and extension.

Perfusion MRI Analysis Toolkit

Clinical pipeline BET-based segmentation Quantitative perfusion

Clinical-facing perfusion MRI pipeline designed under real deployment constraints.
Availability: internal clinical software (not publicly released).

Highlights

  • Problem: reliable perfusion quantification requires robust preprocessing and segmentation under heterogeneous tumour appearances.
  • Contribution: preprocessing modules, BET-based segmentation workflows for complex tumour cases, interactive ROI tools, quantitative perfusion metrics.
  • Evidence: validated metric computation via independent numerical integration (trapezoidal & Simpson’s rules) and cross-checks against existing workflows on clinical cases.

Availability

Not publicly released due to clinical deployment and governance constraints. Documentation/demonstrations can be provided upon request where permitted.

Cardiac Mesh Reconstruction & MedMesh

3D anatomy GNN supervision In-silico trials

Structure-aware template-based deformation for automated 3D cardiac mesh reconstruction, supported by MedMesh (dataset generation & inspection tooling).
Research and software artefacts available; method development ongoing.

Highlights

  • Problem: existing approaches often use CNN features only, which can miss global anatomical context needed for accurate meshes.
  • Contribution: propose GNN-based segmentation supervision to capture local + global context and improve mesh fidelity; implemented supporting dataset/pipeline tooling via MedMesh.
  • Evidence: reproducible software artefacts for dataset generation and inspection; designed for downstream simulation and in-silico trial workflows.
